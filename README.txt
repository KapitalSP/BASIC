# BASIC v1.0

The most fundamental, high-strength skeleton for local AI intelligence. 
**BASIC** is a universal framework designed to host AI engines with zero external dependencies.

###  Structure (Slots)
| Directory | Role |
| :--- | :--- |
| `/models` | The Engine Socket. Drop your LLM files here. |
| `/refinery` | Reserved for data processing and cleaning modules. |
| `/plugins` | Expansion slot for search, vision, or external tools. |
| `/logs` | Local-only activity logging for privacy. |

### **Quick Start**
1. **Initialize:** `python basic.py`
2. **Mount:** Place a `.gguf` model in the `/models` directory.
3. **Execute:** Run `basic.py` again to start your permanent local session.
